# AI Architect Track

This track equips senior engineers and AI architects with the knowledge and skills to design, govern, and scale AI systems across multi‑cloud environments.

## Modules

| Module | Title |
|--------|-------|
| 08.1 | AI System Architecture & Design Patterns |
| 08.2 | MLOps at Scale & Platform Engineering |
| 08.3 | Cloud AI Platform Architecture (Multi-cloud) |
| 08.4 | LLM and LMM Architecture and RAG Systems Design |
| 08.5 | AI Governance, Security & Compliance |
| 08.6 | Performance Engineering & Cost Optimization |
| 08.7 | Data Architecture for AI |
| 08.8 | AI Strategy, Roadmap & ROI |
| 08.9 | Emerging Architectures & Research Translation |
| 08.10 | **Capstone**: Enterprise AI Platform |
| 08.11 | **Capstone**: LLM-Powered Knowledge System |
| 08.12 | **Capstone**: Real-Time Fraud Detection Platform |
| 08.13 | **Capstone**: Multi-Cloud AI Infrastructure |
| 08.14 | **Capstone**: AI Governance & Compliance Program |

Each module contains a `README.md` with detailed topics and deliverables. The five capstones each combine a written architecture document with a live design review.

The track is **stand-alone**; it assumes general AI/ML fundamentals but does not require completing prior tracks.
