## Recommended Order

1. Start with **Interview Classics** (Titanic, House Prices) to build confidence
2. Move to **Classification Systems** (Spam, Churn) to learn evaluation rigor
3. Try **Prediction & Forecasting** for time-series skills
4. Explore **Recommendation** for a different problem paradigm
5. Tackle **Anomaly Detection** for imbalanced/unsupervised methods
6. Finish with **Healthcare** and **Customer Intelligence** for real-world depth
