# Product Image Classification — Problem Statement

## Business Context
E-commerce platforms need to auto-tag product images for search filtering, inventory organisation, and personalised recommendations. Manual tagging at scale is slow and inconsistent.

## Problem Type
Multi-class classification. Supervised learning.

## Success Metrics
- **Accuracy** ≥ 82% on held-out test set
- **Macro F1** ≥ 0.80 across all classes
- **End-to-end latency** < 50 ms per image
