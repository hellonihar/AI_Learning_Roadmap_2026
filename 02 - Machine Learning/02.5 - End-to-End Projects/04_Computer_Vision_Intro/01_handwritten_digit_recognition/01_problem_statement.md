# Handwritten Digit Recognition — Problem Statement

## Business Context
Automated postal mail sorting, cheque processing in banking, and form digitisation. A system that reads handwritten digits reduces manual data entry costs and speeds up document processing pipelines.

## Problem Type
Multi-class classification (10 classes: digits 0–9). Supervised learning.

## Success Metrics
- **Accuracy** ≥ 94% on test set
- **Inference time** < 10 ms per sample
- **Confusion matrix** analysis to identify commonly confused digit pairs (e.g., 4↔9, 7↔1)
