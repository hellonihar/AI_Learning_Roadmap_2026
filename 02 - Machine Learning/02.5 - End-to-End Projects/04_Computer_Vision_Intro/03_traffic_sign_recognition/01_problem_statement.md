# Traffic Sign Recognition — Problem Statement

## Business Context
Autonomous vehicles and driver-assistance systems must recognise traffic signs in real-time from camera feeds. Misclassification can lead to dangerous driving decisions (e.g., Stop vs Speed Limit).

## Problem Type
Multi-class classification (43 classes). Supervised learning.

## Success Metrics
- **Accuracy** ≥ 90% on test set
- **Per-class F1** ≥ 0.85 for all classes
- **Inference time** < 30 ms per frame
