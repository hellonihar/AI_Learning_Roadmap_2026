# Resume Screening — Problem Statement

## Business Context
HR teams receive hundreds of resumes per opening. Automating the screening pipeline — extracting skills, categorising roles, and ranking candidates — cuts time-to-hire by 40%.

## Problem Type
Multi-label text classification (one resume can belong to multiple categories). Supervised learning.

## Success Metrics
- **Macro F1** ≥ 0.85 on held-out test set
- **Per-class F1** ≥ 0.75 for all categories
- **Skill extraction precision** ≥ 90%
