# Sentiment Analysis — Problem Statement

## Business Context
Brand monitoring, customer feedback analysis, and social media listening. Companies need to automatically determine whether a product review or social post is positive, negative, or neutral.

## Problem Type
Binary text classification (positive / negative). Supervised learning.

## Success Metrics
- **Accuracy** ≥ 88% on test set
- **ROC AUC** ≥ 0.93
- **Inference time** < 5 ms per review
