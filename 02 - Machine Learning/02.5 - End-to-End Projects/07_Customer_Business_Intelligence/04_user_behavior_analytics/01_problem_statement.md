# User Behavior Analytics — Problem Statement

## Business Context
A SaaS product has 500K monthly active users but a 12% activation rate (users reaching core "aha moment" within first week). Retention drops from 40% (Month 1) to 18% (Month 6). Need to analyze session behavior, identify where users drop off, and compute engagement scores to target retention campaigns.

## Problem Type
- **Primary**: Descriptive analytics (funnel, cohort, session analysis)
- **Secondary**: Engagement scoring (feature usage aggregation)

## Success Metrics
- **Conversion rate** per funnel step (identify biggest drop-offs)
- **Cohort retention curves** (weekly/daily active users over time)
- **Engagement score distribution** (Heads/Hearts/Tails framework)
- Actionable: number of interventions identified from drop-off analysis
