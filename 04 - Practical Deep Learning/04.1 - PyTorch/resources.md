## Books

- **Dive into Deep Learning (d2l.ai)** — [d2l.ai](https://d2l.ai/) — Interactive textbook with PyTorch, JAX, and TensorFlow code. Covers fundamentals through transformers, GANs, and recommendation systems.
- **Deep Learning with PyTorch** (Stevens, Antiga, Viehmann) — [pytorch.org/assets/deep-learning-with-pytorch](https://pytorch.org/assets/deep-learning-with-pytorch/) — Manning Publications, official PyTorch book.
- **Programming PyTorch for Deep Learning** (Ian Pointer) — O'Reilly Media. Practical focus on production deployment.
- **Natural Language Processing with PyTorch** (Rao, McMahan) — O'Reilly Media. NLP-focused PyTorch guide.

## Courses

- **fast.ai Practical Deep Learning for Coders** — [course.fast.ai](https://course.fast.ai/) — Top-down approach (train models immediately, theory later). Uses PyTorch as the backend. Free.
- **Stanford CS231n: CNNs for Visual Recognition** — [cs231n.stanford.edu](http://cs231n.stanford.edu/) — PyTorch assignments in recent years.
- **Stanford CS224n: NLP with Deep Learning** — [web.stanford.edu/class/cs224n](https://web.stanford.edu/class/cs224n/) — PyTorch-based NLP assignments.
- **NYU Deep Learning (Yann LeCun / Alfredo Canziani)** — [cdl.nyu.edu/deep-learning](https://cdl.nyu.edu/deep-learning/) — Covers PyTorch fundamentals in depth.
- **Full Stack Deep Learning** — [fullstackdeeplearning.com](https://fullstackdeeplearning.com/) — Covers the entire ML lifecycle with PyTorch.
- **PyTorch for Deep Learning (Zero to Mastery)** — Daniel Bourke's free course on YouTube. Comprehensive from beginner to advanced.
- **Alfredo Canziani's PyTorch Tutorials** — [github.com/Atcold/pytorch-Deep-Learning](https://github.com/Atcold/pytorch-Deep-Learning/) — NYU course materials with Jupyter notebooks.

## Tools & Utilities

- **TorchMetrics** — Standard metrics (accuracy, F1, AUROC) — [torchmetrics.readthedocs.io](https://torchmetrics.readthedocs.io/)
- **TorchLens** — Visualize neural network internals — [github.com/torchlens](https://github.com/johnmarktaylor91/torchlens)
- **Torchinfo (formerly torchsummary)** — Parameter count & layer summary — `pip install torchinfo`
- **einops** — Readable tensor operations (rearrange, repeat, reduce) — [github.com/arogozhnikov/einops](https://github.com/arogozhnikov/einops)

> **Getting started:** Work through the official 60-minute blitz, then pick a project. The fastest way to learn PyTorch is to train a model on a real dataset.
