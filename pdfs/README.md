# PDFs

Place PDF files in this folder, then open `viewer.html?file=your-file.pdf` in a browser.

## Usage

1. Drop your PDF into this folder
2. Open `viewer.html?file=your-file.pdf` — or rename `viewer.html` to match your file

## View on GitHub

GitHub renders PDFs directly. Link to any PDF in this repo and GitHub will display it inline:

```markdown
[View PDF](./1784453327269.pdf)
```
