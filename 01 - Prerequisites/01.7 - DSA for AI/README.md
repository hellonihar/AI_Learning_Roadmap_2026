# DSA for AI

Data Structures & Algorithms fundamentals essential for AI engineering — coding interviews, efficient ML implementations, and problem-solving.

## How to use

Work through sequentially. Each section builds on previous concepts.

## Checklist

- [ ] Big-O notation: time & space complexity
- [ ] Arrays: traversal, prefix sums, sliding window
- [ ] Strings: character frequency, two pointers, anagrams
- [ ] Linked lists: fast & slow pointer, cycle detection
- [ ] Stacks: valid parentheses, monotonic stack
- [ ] Queues: deque, circular queue, sliding window
- [ ] Binary search + binary search on answer
- [ ] Sorting: merge sort, quick sort, counting sort
- [ ] Recursion: recursion tree, backtracking
- [ ] Hashing: frequency maps, optimized lookups
- [ ] Problem patterns: sliding window, two pointers, prefix sums
- [ ] Trees: traversals, BST, N-ary, tree DP
- [ ] Graphs: BFS, DFS, shortest path, union find, MST
