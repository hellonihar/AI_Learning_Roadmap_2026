# AI Coding

Learn to use AI assistants effectively for writing, debugging, refactoring, and understanding code — with awareness of their limitations.

## How to use

Work through sequentially. Each section builds on the previous.

## Checklist

- [ ] Understand what AI coding assistants are and how they differ
- [ ] Code generation vs code completion
- [ ] Write clear, contextual coding prompts
- [ ] Iteratively refine AI output
- [ ] Validate AI-generated code for correctness and security
- [ ] Use AI for debugging and error explanation
- [ ] Refactor code with AI assistance
- [ ] Use AI to navigate and understand unfamiliar codebases
- [ ] Generate ML/data preprocessing pipelines with AI
- [ ] Write automation scripts with AI
- [ ] Know the major AI coding tools (Copilot, Claude, Cursor, etc.)
- [ ] Recognize limitations: hallucinated APIs, outdated patterns, overconfidence
