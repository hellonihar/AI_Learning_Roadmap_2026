# What Are AI Coding Assistants?

AI coding assistants are tools that use large language models (LLMs) to help you write, understand, and debug code. They are trained on vast amounts of public code and documentation.

## How They Work

You provide a prompt (a description of what you want), and the AI generates code, explanations, or suggestions based on patterns it learned during training.

## Major Assistants

| Tool | Type | Key Feature |
|---|---|---|
| **GitHub Copilot** | IDE plugin | Inline completions, chat in editor |
| **Claude** (Anthropic) | Chat + API | Long context, strong reasoning |
| **ChatGPT** (OpenAI) | Chat | Broad knowledge, code generation |
| **Cursor** | AI-native IDE | Understands full codebase context |
| **Codeium** | IDE plugin | Free tier, multi-language |
| **Continue** | Open source IDE plugin | Local models, customizable |
| **AWS CodeWhisperer** | IDE plugin | AWS-centric, free |

## What They Can Do

- Generate functions, classes, tests, and scripts from descriptions
- Explain unfamiliar code line by line
- Refactor code (rename, extract, simplify)
- Debug errors and suggest fixes
- Write documentation and docstrings
- Translate code between languages
- Generate SQL queries, regex patterns, shell commands
