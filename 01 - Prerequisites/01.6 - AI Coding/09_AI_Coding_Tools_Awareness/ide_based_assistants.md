# IDE-Based AI Assistants

AI assistants integrated directly into your code editor.

## GitHub Copilot

The most popular AI coding assistant. Works inside VS Code, JetBrains, Neovim, and more.

| Feature | Capability |
|---|---|
| Completions | Suggests code as you type (Tab to accept) |
| Chat | Open AI chat within editor |
| Inline | Select code → "Fix this" / "Explain" |
| Agent mode | Full file and project-level generation |

**Best for**: developers who want inline completions without leaving their editor.

**Cost**: Free for students/OSS maintainers, $10-19/month otherwise.

## Cursor

An AI-native IDE built on VS Code with deeper codebase understanding.

| Feature | Capability |
|---|---|
| Full file editing | AI rewrites entire functions/files |
| Codebase-wide context | Understands all files in tab history |
| Terminal integration | Ask AI about errors, it reads terminal output |
| Chat with codebase | "Where is the training loop?" across entire project |

**Best for**: developers who want AI to understand the full codebase, not just the current file.

**Cost**: $20/month.

## Codeium / Windsurf

Free alternative to Copilot.

| Feature | Capability |
|---|---|
| Completions | Similar to Copilot |
| Chat | Built-in AI chat |
| Multi-language | 70+ languages |
| Free tier | Generous free tier |

**Best for**: individuals or teams who want AI assistance without paying per seat.

## Continue

Open-source AI code assistant. You choose the model (local or cloud).

**Best for**: developers who want full control, privacy, and use local LLMs.
