# SQL for AI Development

SQL is the backbone of data extraction, feature engineering, and analytics in AI workflows. These lessons focus on the query patterns you'll actually use day-to-day.

## How to use

Work through `01_Core_Queries` to `06_AI_SQL_Patterns` sequentially. Each builds on the last.

## Checklist

- [ ] SELECT, WHERE, ORDER BY, LIMIT
- [ ] Aggregate functions & GROUP BY
- [ ] INNER, LEFT, RIGHT, FULL JOINs
- [ ] Subqueries (correlated & uncorrelated)
- [ ] CTEs with WITH
- [ ] Window functions: ROW_NUMBER, RANK, LAG, LEAD
- [ ] SUM / AVG OVER with PARTITION BY
- [ ] INSERT, UPDATE, DELETE, MERGE
- [ ] Transactions (BEGIN, COMMIT, ROLLBACK)
- [ ] CREATE TABLE, constraints, indexes
- [ ] Normalization basics
- [ ] JSON queries in PostgreSQL
- [ ] Sampling strategies
- [ ] Time-based window aggregations for features
- [ ] Deduplication patterns
- [ ] Execution plan basics for performance
