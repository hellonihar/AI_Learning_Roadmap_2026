# What is Cloud Computing?

Cloud computing is on-demand delivery of compute, storage, and networking over the internet with pay-as-you-go pricing.

## Key Benefits
- **No upfront capital** — pay only for what you use
- **Elasticity** — scale up/down instantly
- **Global reach** — deploy in regions worldwide
- **Managed services** — focus on code, not servers

## For AI Engineers
- Train models on 8× H100 GPUs for a few hours, then shut down
- Store TBs of datasets in object storage, accessed globally
- Deploy inference endpoints with auto-scaling
- Use managed ML services instead of building infra from scratch
