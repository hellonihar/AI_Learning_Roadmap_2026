# Cloud Fundamentals for AI Engineers

Understand cloud computing concepts, core services, and how to design AI workloads on the cloud.

## How to use

Start with `01_Cloud_Basics` for foundational concepts, then explore sections based on your need. `09_Cloud_for_AI` is the most relevant for AI workloads.

## Checklist

- [ ] Cloud service models: IaaS, PaaS, SaaS
- [ ] Public vs private vs hybrid cloud
- [ ] Regions and availability zones
- [ ] Shared responsibility model
- [ ] Compute: VMs, containers, serverless
- [ ] Storage: object, block, file
- [ ] Networking: VPC, subnets, security groups, DNS
- [ ] Databases: relational vs NoSQL, managed services
- [ ] IAM: users, roles, policies
- [ ] Docker fundamentals and registries
- [ ] GPU instances and AI deployment patterns
- [ ] Pay-as-you-go pricing and cost management
- [ ] Infrastructure as code and CI/CD concepts
- [ ] AWS / GCP / Azure awareness
