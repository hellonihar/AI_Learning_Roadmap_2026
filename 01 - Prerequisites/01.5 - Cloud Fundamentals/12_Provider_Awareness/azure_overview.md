# Microsoft Azure Overview

## Key Services for AI
| Category | Service |
|---|---|
| Compute | VMs (GPU: ND H100, NCas T4), AKS, Container Apps, Functions |
| Storage | Blob Storage, Managed Disks, Files |
| ML Platform | Azure ML (training, endpoints, automated ML) |
| AI Services | OpenAI Service (GPT-4), Cognitive Services (vision, speech, language) |
| Databases | Azure SQL, Cosmos DB, Cache for Redis |
| Networking | VNet, Load Balancer, DNS, Front Door |
| Security | Entra ID (IAM), Key Vault, Defender for Cloud |
| CI/CD | Azure DevOps, GitHub Actions |

## Strengths
- **OpenAI Service**: first-class managed OpenAI (GPT-4, GPT-4o) with enterprise security
- **Enterprise integration**: deep integration with Active Directory, Office 365, Dynamics
- **Hybrid cloud**: Azure Arc, Azure Stack for on-premises
- **AI Studio**: unified platform for building, evaluating, and deploying AI apps
- **Copilot ecosystem**: extensibility for Microsoft 365

## Considerations
- Some services have fewer features than AWS equivalents
- Naming can be confusing (many similar service names)
- Portal UX can be slower than competitors
