# Variables, Data Types, Operators

## Basic Types
```python
a: int = 42
b: float = 3.14
c: str = "hello"
d: bool = True
e: list = [1, 2, 3]
f: tuple = (1, 2)
g: dict = {"key": "value"}
h: set = {1, 2, 3}
```

## Type Conversion
```python
int("42"); float("3.14"); str(42)
list((1,2,3)); tuple([1,2,3])
```

## Operators
```python
+ - * / // % **    # arithmetic
== != < > <= >=    # comparison
and or not          # logical
& | ^ ~ << >>      # bitwise
is in               # identity / membership
```

## String Methods
```python
s.upper(); s.lower(); s.strip(); s.split(",")
",".join(["a","b"]); s.replace("old","new")
s.startswith("x"); s.endswith("y"); "sub" in s
```
