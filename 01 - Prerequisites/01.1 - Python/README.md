# Python for AI Engineering

A curated progression from Python fundamentals through the AI/ML stack and engineering practices needed to build production AI systems.

## How to use

Each folder builds on the previous. Start with **01_Basics** and work through sequentially. By the end you'll have built three real projects.

## Checklist

- [ ] Data types, control flow, functions
- [ ] Comprehensions, lambdas, decorators
- [ ] Error handling & logging
- [ ] Classes, inheritance, dunder methods
- [ ] NumPy: arrays, broadcasting, linear algebra
- [ ] Pandas: DataFrames, groupby, merge
- [ ] Matplotlib / Seaborn basics
- [ ] Scikit-learn: estimator API, pipelines
- [ ] PyTorch: tensors, autograd, nn.Module
- [ ] Hugging Face transformers
- [ ] LangChain basics
- [ ] Type hints & mypy
- [ ] Async / await
- [ ] Pydantic models
- [ ] FastAPI services
- [ ] pytest for ML pipelines
