# Software Essentials for AI Engineers

Covers version control, development environment setup, and API integration — the daily tools you need to build and ship AI systems.

## How to use

Work through in order. `01_Version_Control` and `02_Development_Environment` are prerequisites for everything else.

## Checklist

- [ ] git init, clone, add, commit, push, pull
- [ ] Branching, merging, resolving conflicts
- [ ] Pull request workflow (fork, PR, code review)
- [ ] .gitignore, git stash, cherry-pick, rebase
- [ ] Terminal basics (env vars, PATH, aliases)
- [ ] Python virtual environments (venv, uv)
- [ ] VS Code setup for AI development
- [ ] Dependency management (requirements.txt, pyproject.toml)
- [ ] Docker: Dockerfile, docker-compose, dev containers
- [ ] HTTP methods, status codes, rate limits
- [ ] REST API calls with requests / httpx
- [ ] OpenAI API: chat completions, streaming, retries
- [ ] Async API calls with asyncio
- [ ] Basic API design with FastAPI
